# Healthcare Workers Mental Health Risk Prediction

## Project Overview
This project builds an automated risk-identification system to predict mental health challenges among healthcare professionals. By processing a robust database of 5,000 healthcare records, the system evaluates multiple classification algorithms to identify the most accurate predictive model for clinical and administrative use.

## Technical Stack
* **Language:** Python
* **Libraries & Tools:** Scikit-learn, Pandas, NumPy
* **Algorithms:** Logistic Regression, Random Forest, XGBoost
* **Techniques:** Exploratory Data Analysis (EDA), Predictive Modeling

## Repository Contents
* `Project_CIS550.ipynb`: The core Python notebook containing data cleaning, exploratory analysis, model training, and comparative evaluation.
* `Project Reports (PDFs)`: Comprehensive academic and technical documentation detailing the statistical findings and model architecture.
* `Presentation (PPTX)`: A high-level slide deck summarizing the project's methodology and key clinical insights.

## Methodology & Results
1. **Data Processing:** Cleaned and standardized a dataset of 5,000 healthcare worker records, conducting rigorous EDA to isolate critical features.
2. **Model Evaluation:** Programmed, tested, and compared three distinct classification algorithms (Logistic Regression, Random Forest, XGBoost) to determine optimal performance.
3. **Key Findings:** Identified stress level (r=0.74) and workplace environment (r=-0.55) as the strongest risk predictors.
4. **System Accuracy:** The final optimized Logistic Regression model achieved a highly reliable 94.1% accuracy rate in risk identification.
